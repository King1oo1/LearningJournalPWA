Week 2: Frontend Fundamentals & Mobile-First Design

January 22, 2024 Frontend

How did you approach mobile-first design?

I started by designing for the smallest screens first, using a single-column layout and ensuring all interactive elements were touch-friendly. I implemented CSS Grid and Flexbox for the core layout structure, then used media queries with min-width breakpoints to progressively enhance the design for larger screens.

What was the most useful HTML or CSS concept you applied this week?

The most valuable concept was CSS Grid for creating responsive layouts without media queries. Using grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)) allowed me to create flexible card layouts that automatically adapt to different screen sizes. CSS custom properties (variables) were also extremely useful for maintaining consistent spacing and colors.

What part of HTML or CSS did you find most challenging or confusing?

The most challenging aspect was managing the navigation menu's transition from mobile hamburger style to desktop horizontal layout. Ensuring consistent spacing and alignment across different browsers and devices also required careful testing. CSS specificity was sometimes confusing when trying to override default styles.
