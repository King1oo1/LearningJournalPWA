Week 2: Frontend Fundamentals

Posted on: October 2025

Mobile-First Design Approach

I started by designing for mobile devices first, using a single-column layout that naturally expands on larger screens. This approach ensured that the core content and functionality work perfectly on smaller viewports.

Most Useful HTML/CSS Concept

The most useful concept I applied was CSS Flexbox for creating responsive navigation and layout structures. Flexbox made it easy to create flexible components that adapt to different screen sizes without complex calculations.

Most Challenging Aspect

I found media queries and breakpoint selection to be the most challenging. Determining the optimal breakpoints for different devices required testing and adjustment to ensure smooth transitions between layouts.
